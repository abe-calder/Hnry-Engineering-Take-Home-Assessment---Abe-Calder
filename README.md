# Getting Started

You will need the following software installed on your machine:
## Node.js
https://nodejs.org/en/download/current
## npm
https://docs.npmjs.com/downloading-and-installing-node-js-and-npm

# Installation

Clone the repository:

Bash
```
git clone git@github.com:abe-calder/Hnry-Engineering-Take-Home-Assessment---Abe-Calder.git

cd Hnry-Engineering-Take-Home-Assessment---Abe-Calder

npm install
```

# To View in the brower and use the Tax to pay Calculator

`npm run dev`
